# Portafolio Profesional - Roman Velasco

![Estado del Proyecto](https://img.shields.io/badge/Estado-Terminado-success?style=flat-square)
![Tecnologías](https://img.shields.io/badge/Tecnologías-HTML%20%7C%20CSS%20%7C%20JS-blue?style=flat-square)

Un portafolio web moderno y profesional diseñado para destacar proyectos de ingeniería en sistemas y desarrollo web. Construido con un enfoque en **estética premium**, **micro-interacciones** y **rendimiento**.

## 🚀 Características

- **Diseño Moderno & Oscuro**: Interfaz elegante con paleta de colores "Navy/Slate" y acentos neón.
- **Efectos Visuales**: Glitch effect en el título, cursor personalizado interactivo y animaciones de entrada al hacer scroll.
- **Glassmorphism**: Barra de navegación con efecto de desenfoque (backdrop-filter).
- **Totalmente Responsivo**: Adaptado para móviles, tablets y escritorio.
- **Zero-Dependency Bundle**: Todo el código (HTML, CSS, JS) está consolidado en un único archivo `index.html` para facilitar el despliegue en cualquier hosting gratuito.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Variables CSS, Flexbox, Grid, Animaciones (@keyframes) y Media Queries.
- **JavaScript (ES6+)**: IntersectionObserver para animaciones de scroll y lógica de interactividad.

## 📂 Estructura del Proyecto

Actualmente, el proyecto está optimizado para producción en un solo archivo:

```
/
└── index.html  <-- Contiene toda la lógica, estilos y estructura.
```

> **Nota**: Las imágenes se cargan desde URLs externas o carpetas locales (ej. `images/`) según la configuración.

## 📦 Instalación y Despliegue

Este proyecto está diseñado para ser "Drag & Drop".

1.  **Descarga** el archivo `index.html`.
2.  **Sube** el archivo a la carpeta raíz de tu hosting (public_html, htdocs, etc.).
3.  ¡Listo! Tu sitio estará en línea inmediatamente.

**Hosting recomendado**: InfinityFree, GitHub Pages, Vercel, o Netlify.
